# Startup Investment Analysis

## 📌 Project Overview
This project analyzes startup funding data to identify investment trends, sector performance, and factors influencing startup success.

## 📊 Objectives
- Analyze funding distribution across sectors
- Identify top investors and startup hubs
- Understand startup success patterns
- Build an interactive dashboard

## 🛠 Tools Used
- Python (Pandas)
- Power BI

## 📁 Dataset
The dataset includes:
- Company Name
- Sector
- Headquarters
- Funding Amount
- Funding Round
- Lead Investor

## 📊 Key Insights
- Technology sector receives the highest funding
- Investment is concentrated in major cities
- Certain investors dominate funding rounds
- Early-stage funding is most common

## 📈 Dashboard
The Power BI dashboard includes:
- Funding by Sector
- Top Cities
- Top Investors
- Funding Round Distribution
- KPI Metrics

## 📄 Deliverables
- Power BI Dashboard (.pbix)
- Investor Trends Report (PDF)
- Founder Success Summary (PDF)

## 🚀 Conclusion
The project highlights key investment trends and startup success patterns, providing insights into how the startup ecosystem operates.
